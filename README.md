# Foooly Filekit
O **Filekit** usa plugins externos para baixar o conteúdo.
- Necessário:<br>
  [ytdl (Clique para baixar)](https://github.com/yt-dlp/yt-dlp)<br>
- Opcional*:<br>
  [ffmpeg (Clique para baixar)](https://www.gyan.dev/ffmpeg/builds/#release-builds)<br><br>
_*Para baixar vídeos do youtube, é RECOMENDADO a instalação do FFMPEG para que funcione corretamente._

## Como baixar?
Acesse **configurações** do Filekit e clique em **Anexar plugins**, e anexe o **ytdl.exe** e **ffmpeg.exe** na pasta **plugins** que abriu.<br>
Depois disso, só colar a url da plataforma desejada e clique no botão **baixar** e pronto!
