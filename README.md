## ⚡ Plataformas compatíveis

🎥 **Vídeos & Áudios**  
- YouTube (Vídeo/Áudio), Youtube Music (Apenas áudio)
- Instagram (Vídeo/Áudio)
- Kick* (Vídeo/Áudio)
- Twitch (Vídeo/Áudio)
- Vimeo (Vídeo/Áudio)
- TikTok *(com cookies -- você precisa ter o **tiktok_cookies.txt** anexado na pasta **plugins**)*

_*No momento no Filekit só funciona pra baixar clipes, mas não se preocupe, use o Plugins Built-In da Kick._

## 🔧 Plugins Built-In

Os **Plugins Built-In** são sites que estão imbutidos no Filekit sem que você precise sair do app.

Disponíveis: (Podem ser adicionados mais no futuro)
- Spotify *(via SpotiDownloader)*  
- Deezer *(via Deezmate)*
- Kick *(Via kick-video.download)*
- Tiktok *(Via ssstik.io/pt)*

## 🚀 Como usar

1. **Baixe o instalador** na aba [📦 Releases](https://github.com/jhoOrdann/filekit/releases)
2. Execute e instale o Filekit.
3. Abra o app e cole o link do vídeo ou música.
4. Escolha:
   - Plataforma
   - Qualidade
   - Codec
   - Tipo de arquivo (`.mp4`, `.mkv`, `.mp3`, etc)
   - Pasta de destino
   - Nome do arquivo (opcional)
5. Clique em **Baixar** e pronto ✅

## 🧩 Plugins necessários

Para o Filekit funcionar corretamente, você precisa adicionar alguns **plugins externos** (ferramentas usadas pelo programa).

### 📁 Onde ficam?
Acesse pelo próprio app:  
> ⚙️ Configurações → **Adicionar Plugins**

ou manualmente:
> C:\Users\{usuário}\AppData\Roaming\Filekit\plugins\

### 🔧 Baixe os seguintes arquivos:

| Plugin | Descrição | Link oficial |
|---------|------------|--------------|
| `yt-dlp.exe` | Baixador de vídeos principal | [yt-dlp GitHub](https://github.com/yt-dlp/yt-dlp/releases/latest) |
| `ffmpeg.exe` | Utilitário para converter/mesclar áudio e vídeo | [FFmpeg Builds](https://www.gyan.dev/ffmpeg/builds/) |
| `tiktok_cookies.txt` *(opcional)* | Arquivo de cookies para TikTok | Faça login no TikTok → exporte cookies via extensão “Get cookies.txt” |

📌 Coloque todos esses arquivos dentro da pasta `plugins/` criada automaticamente na primeira execução.
